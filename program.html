Symposium Program

The REDCap Africa Symposium program includes the main symposium, advanced training, introductory training, and system administrator sessions.

Advanced Workshop

19-20 October | 09:00 - 17:00

Experienced REDCap users will explore branching logic, calculated fields, API integrations and advanced workflows.

Symposium

21-22 October | 08:00 - 17:00

Keynote sessions, panel discussions, and abstract presentations that showcase REDCap innovations across African research.

Introductory Workshop

21-23 October | 09:00 - 17:00

Beginner-friendly training on REDCap project setup, data entry, and essential tools for new users.

System Administrator Training

23 October | 09:00 - 17:00

Practical guidance on server setup, security, maintenance and troubleshooting for REDCap sys-admins.

Program Notes

The Symposium is the core event. Training workshops are supplementary and require Symposium registration to attend.
